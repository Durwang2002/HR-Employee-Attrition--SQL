# 👨‍💼 HR Employee Attrition Analysis using SQL

## 📌 Project Overview

This project analyzes employee attrition using SQL to identify the key factors influencing employee turnover. The analysis focuses on workforce demographics, employee satisfaction, salary, overtime, work-life balance, and department-wise performance.

The project demonstrates end-to-end SQL analysis suitable for Data Analyst portfolios and technical interviews.

---

## 🎯 Business Objective

The objective of this project is to:

- Analyze employee attrition
- Measure HR KPIs
- Identify high-risk employee groups
- Support employee retention strategies
- Improve workforce planning through SQL analysis

- **Domain:** Human Resources
- **Dataset:** HR Employee Attrition
- **Database:** `hr_employee_attrition`
- **Table:** `hr-employee-attrition`
